    дёҖгҖҒе·ҘдёҡзӣёжңәжҲҗеғҸзҡ„е®һйҷ…йңҖжұӮ

    е·ҘдёҡзӣёжңәеҸҲз§°ж‘„еғҸжңәпјҢзӣёжҜ”дәҺдј з»ҹзҡ„ж°‘з”ЁзӣёжңәпјҲж‘„еғҸжңәпјүиҖҢиЁҖпјҢе®ғе…·жңүй«ҳзҡ„еӣҫеғҸзЁіе®ҡжҖ§гҖҒй«ҳдј иҫ“иғҪеҠӣе’Ңй«ҳжҠ—е№Іжү°иғҪеҠӣзӯүпјҢзӣ®еүҚеёӮйқўдёҠзҡ„е·ҘдёҡзӣёжңәеӨ§еӨҡжҳҜеҹәдәҺ CCDпјҲCharge CoupledDeviceпјүжҲ– CMOSпјҲComplementary MetalOxide SemiconductorпјүиҠҜзүҮзҡ„зӣёжңәгҖӮ

    е·ҘдёҡзӣёжңәжҲҗеғҸжңүй»‘зҷҪпјҲgmпјүдёҺеҪ©иүІпјҲgcпјүдёӨз§ҚпјҢеҲҶеҲ«дёәдёӨз§ҚдёҚеҗҢзҡ„еғҸзҙ еӯҳеӮЁж јејҸгҖӮ

    зӣёжҜ”иҫғдәҺиҮӘз„¶еӣҫеғҸеҹәжң¬дёәеҪ©иүІеӣҫеғҸпјҢе·ҘдёҡзӣёжңәжҲҗеғҸдёӯзҒ°еәҰеӣҫдҫқз„¶еҚ жңүеҫҲеӨ§жҜ”дҫӢгҖӮ

    еҜ№дәҺзҒ°еәҰеӣҫеғҸеҒҡеӣҫеғҸеҲҶзұ»д»»еҠЎпјҢжҲ‘д»¬еҫҖеҫҖдјҡйҮҮз”ЁдёҖдәӣжҲҗзҶҹзҡ„жЁЎеһӢжһ¶жһ„пјҢжҜ”еҰӮresnetгҖҒvggзӯүгҖӮ

    е®һи·өиҜҒжҳҺпјҢImagenetйў„и®ӯз»ғеҜ№е·Ҙдёҡдә§е“Ғзҡ„зҒ°еәҰеӣҫеғҸдҫқз„¶жңүиҝҒз§»еӯҰд№ ж•ҲжһңпјҢйӮЈд№ҲйҮҮз”ЁеҚ•йҖҡйҒ“зҒ°еәҰеӣҫзӣёйҮҮз”ЁImagenetйў„и®ӯз»ғе°ұеҝ…з„¶йңҖиҰҒйҮҮз”Ёе…¶еҜ№еә”зҡ„еӣҫзүҮйў„еӨ„зҗҶж–№ејҸпјҢеҚіе°ҶеӣҫзүҮжү©е……дёәдёүйҖҡйҒ“з„¶еҗҺеҮҸеҺ»еқҮеҖјйҷӨд»Ҙж ҮеҮҶе·®пјҢеҸҜжҳҜзҒ°еәҰеӣҫеғҸжҳҜеҚ•йҖҡйҒ“зҡ„е‘ҖпјҢжҳҜеҗҰжңүдёҖз§Қж–№жі•жқҘеҜ№жЁЎеһӢз»“жһ„еҒҡдёҖж¬Ўзӯүд»·д»ЈжҚўд»ҺиҖҢдҪҝеҫ—зҺ°жңүжЁЎеһӢзӣҙжҺҘд»ҺеҚ•йҖҡйҒ“еӣҫеғҸиҝӣиЎҢи®Ўз®—пјҢиҖҢдёҚйңҖиҰҒе°ҶеҚ•йҖҡйҒ“еӨҚеҲ¶дёәдёүйҖҡйҒ“е‘ўпјҹ

    дәҢгҖҒе…¬ејҸжҺЁеҜј

    е…¶е®һеңЁи§Ҷйў‘зҗҶи§Јд»»еҠЎдёӯпјҢеёёеёёйҮҮз”Ёжү©е……Imagenetйў„и®ӯз»ғ2DеҚ·з§Ҝж ёзҡ„еҠһжі•пјҢе°Ҷ2DеҚ·з§Ҝжү©е……дёә3DеҚ·з§ҜпјҢжңҖз»Ҳзӣ®ж ҮжҳҜдёәдәҶе®һзҺ°еңЁиҫ“е…ҘзӣёеҗҢNдёӘеӣҫзүҮз»„жҲҗзҡ„дёҖдёӘSequenceдёӯпјҢеҫ—еҲ°NдёӘдёҺеҺҹжқҘ2D Imagenetйў„и®ӯз»ғжҺЁзҗҶеҮәзҡ„зӣёеҗҢзҡ„зү№еҫҒеӣҫгҖӮ

    зұ»жҜ”дәҺдёҠйқўзҡ„еҺҹзҗҶпјҢеңЁжІЎжңүеҮҸеҺ»еқҮеҖјйҷӨд»Ҙж ҮеҮҶе·®зҡ„ж“ҚдҪңж—¶пјҢеӣ дёәеҚ•йҖҡйҒ“еӣҫеғҸдјҡзңӢеҒҡеҗ„дёӘйҖҡйҒ“еҖјйғҪзӣёеҗҢзҡ„дёүйҖҡйҒ“еӣҫеғҸпјҢжҲ‘д»¬д№ҹеҸҜд»ҘеҫҲиҮӘз„¶зҡ„е°Ҷ3йҖҡйҒ“иҫ“е…Ҙзҡ„еҚ·з§Ҝж ёпјҢеңЁиҫ“е…ҘйҖҡйҒ“дёҠзҙҜеҠ пјҢе®һзҺ°еҜ№еҚ•йҖҡйҒ“еӣҫзүҮзҡ„еҚ·з§Ҝж“ҚдҪңгҖӮ

    и®ҫиҫ“е…ҘеӣҫеғҸдёә \(\rm{X} \in R^{3,H,W}\)пјҢXзҡ„дёүдёӘйҖҡйҒ“ж»Ўи¶і\(\rm{X_r} = \rm{X_g} = \rm{X_b} = \rm{X_{gray}} \in R^{H,W}\)гҖӮ\(\rm{X_{gray}}\)иЎЁзӨәзҒ°еәҰеӣҫеғҸгҖӮи®ҫеҚ·з§Ҝж ёжқғйҮҚдёә\(\rm{W} \in R^{C_{out}, C_{in}, K_h, K_w}\)пјҢbiasдёә\(\rm{b} \in R^{C_{out}}\)гҖӮе…¶дёӯ\(\rm{C}_{out}\)дёәиҫ“е…ҘйҖҡйҒ“ж•°пјҢ\(\rm{C}_{in}\)дёәиҫ“е…ҘйҖҡйҒ“ж•°пјҢжӯӨж—¶\(\rm{C}_{in}=3\)пјҢ\(\rm{K}_h, \rm{K}_w\)дёәеҚ·з§Ҝж ёзҡ„й«ҳе’Ңе®ҪпјҢ\(*\) иЎЁзӨәеҚ·з§Ҝж“ҚдҪңпјҢдёҚиҖғиҷ‘batchз»ҙеәҰпјҢиҫ“еҮә\(\rm{output} \in R^{C_{out},H',W'}\)

    еҜ№иҫ“е…ҘеӣҫеғҸзӣҙжҺҘеҒҡеҚ·з§Ҝж“ҚдҪңзҡ„е…¬ејҸеҰӮдёӢпјҢ

    \[ \begin{aligned} \rm{output} &= \rm{W} * \rm{X} + \rm{b} \\ &= \rm{reduce\_sum}(\rm{W}, \rm{axis}=1, \rm{keepdim}=true) * \rm{X_{gray}} + \rm{b} \end{aligned} \]

    е…¶дёӯ\(\rm{reduce\_sum}\)иЎЁзӨәеҜ№еј йҮҸзү№е®ҡз»ҙеәҰеҪ’зәҰжұӮе’ҢпјҢдёҖиҲ¬еј йҮҸиҝҗз®—еә“йғҪж”ҜжҢҒиҜҘиҝҗз®—гҖӮ

    еҸҜжҳҜеңЁImagenetйў„и®ӯз»ғиҰҒжұӮзҡ„йў„еӨ„зҗҶжқЎд»¶дёӢпјҢеҚійҷӨд»Ҙ255еҗҺпјҢеҮҸеҺ»еқҮеҖјйҷӨд»Ҙж ҮеҮҶе·®пјҢжҲ‘д»¬еҸҲиҜҘеҢ–з®ҖиҜҘиҝҮзЁӢе‘ўпјҹ

    и®ҫImagenetж•°жҚ®йӣҶзҡ„еқҮеҖје’Ңж–№е·®\(\rm{mean} = [0.485, 0.456, 0.406]\)пјҢ\(\rm{std} = [0.229, 0.224, 0.225]\)гҖӮ

    еҚ·з§ҜпјҲеҸӮиҖғim2colпјүе’Ңзҹ©йҳөд№ҳжі•еҸҜд»Ҙзӯүд»·пјҢж»Ўи¶із»“еҗҲеҫӢпјҢеҲҷдёҠиҝ°иҝҮзЁӢеҸҜд»ҘеҶҷеҮәеҰӮдёӢе…¬ејҸпјҡ

    \[ \begin{aligned} \rm{output} &= \rm{W} * \frac{ \frac{\rm{X}}{255} - \rm{mean} }{ \rm{std} } + \rm{b} \\ &= \rm{W} * \frac{ \rm{X} - 255 \times \rm{mean} }{ 255 \times \rm{std} } + \rm{b} \\ &= \frac{\rm{W}}{255 \times \rm{std}} * \rm{X} - \rm{W} * \rm{full\_like}(\rm{X}, \frac{ \rm{mean} }{ \rm{std} }) + \rm{b} \\ &= \rm{reduce\_sum}(\frac{\rm{W}}{255 \times \rm{std}}, \rm{axis}=1, \rm{keepdim}=true) * \rm{X_{gray}} + \rm{b} \\ &\ \ \ \ + \rm{reduce\_mean}(- \rm{W} * \rm{full\_like}(\rm{X}, \frac{ \rm{mean} }{ \rm{std} }), axis=[-1, -2], \rm{keepdim}=false) \end{aligned} \]

    е…¶дёӯ\(\rm{full\_like}(\rm{X}, \frac{ \rm{mean} }{ \rm{std} })\)иЎЁзӨәдёҖдёӘдёҺXеҪўзҠ¶зӣёеҗҢпјҢдё”дёүдёӘйҖҡйҒ“зҡ„еҖјдёә\(\frac{ \rm{mean} }{ \rm{std} }\)зҡ„еј йҮҸгҖӮ \(\rm{reduce\_mean}\)иЎЁзӨәеҜ№еј йҮҸзү№е®ҡз»ҙеәҰиҝӣиЎҢеҪ’зәҰжұӮе№іеқҮгҖӮ

    д»ҺдёҠйқўе…¬ејҸеҸҜд»ҘзңӢеҮәдёҠйқўйў„еӨ„зҗҶзҡ„иҝҮзЁӢеҸҜд»ҘиһҚеҗҲеҲ°еҚ·з§Ҝж ёпјҢж–°зҡ„жқғйҮҚжҳҜ\(\rm{reduce\_sum}(\frac{\rm{W}}{255 \times \rm{std}}, \rm{axis}=1, \rm{keepdim}=true)\)пјҢж–°зҡ„biasдёә\(\rm{b} + \rm{reduce\_mean}(- \rm{W} * \rm{full\_like}(\rm{X}, \frac{ \rm{mean} }{ \rm{std} }), axis=[-1, -2], \rm{keepdim}=false)\) гҖӮеҰӮдҪ•зҗҶи§ЈbiasйЎ№дёӯзҡ„еҗҺйЎ№е‘ўпјҹиҜ»иҖ…еҸҜд»ҘиҜ•жғідёҖдёӢпјҢеңЁеҚ·з§Ҝж ёеҸӮж•°дёҚеҸҳзҡ„жғ…еҶөдёӢпјҢиҫ“е…Ҙеј йҮҸзҡ„жҜҸдёӘйҖҡйҒ“еҶ…йғЁйғҪжҳҜзӣёеҗҢзҡ„еҖјпјҢйӮЈд№ҲжҳҜдёҚжҳҜж„Ҹе‘ізқҖиҫ“еҮәйҖҡйҒ“зҡ„еҶ…йғЁпјҢжҜҸдёӘиҫ“еҮәйҖҡйҒ“еҶ…йғЁеҖјд№ҹжҳҜдёҖж ·зҡ„пјҢжӯӨж—¶еҜ№е…¶жұӮmeanе®һйҷ…дёҠзӯүд»·дәҺеҲ©з”Ёеј йҮҸиҝҗз®—еә“дёӯзҡ„е№ҝж’ӯжңәеҲ¶пјҢе°ҶдёҖдёӘйҖҡйҒ“еҶ…йғЁе…ғзҙ зӣёеҗҢзҡ„еј йҮҸиҪ¬жҚўдёәдёҖдёӘеҗ‘йҮҸиЎЁзӨәпјҢиҖҢиҝҷдёӘеҗ‘йҮҸеҪўзҠ¶жҒ°еҘҪдёҺbiasзӣёеҗҢпјҢд»ҺиҖҢеҸҜд»Ҙе®һзҺ°дёҺеҺҹжқҘзҡ„biasзӣёеҠ гҖӮ

    йҒ—жјҸз»ҶиҠӮпјҡ

    йҖҡеёёstage1зҡ„еҚ·з§Ҝж“ҚдҪңйғҪжңүpaddingпјҢдё”padding_value=0пјҢжүҖд»ҘеңЁиҫ№з•ҢеӨ„еҫ—еҲ°зҡ„еҖјдёҚж»Ўи¶ідёҠиҝ°жқЎд»¶пјҢеҚіеңЁйҖҡйҒ“еҶ…йғЁпјҢиҫ№з•ҢеӨ„зҡ„еҖјдёҺеҶ…йғЁзҡ„еҖјдёҚдёҖж ·пјҲиҫ“е…Ҙиҫ“еҮәйғҪжҳҜдёҖж ·пјүпјҢеңЁи®Ўз®—зҡ„ж—¶еҖҷжҲ‘д»¬иҰҒеҝҪз•ҘжҺүpaddingзҡ„и®Ўз®—з»“жһң

    еңЁи®Ўз®—\(- \rm{W} * \rm{full\_like}(\rm{X}, \frac{ \rm{mean} }{ \rm{std} }) + \rm{b}\)жңүдёҖдёӘжҠҖе·§пјҢеҚіе°ҶXи®ҫзҪ®дёәе…Ё0еҗ‘йҮҸпјҢеёҰе…ҘеҲ°\(\rm{W} * \frac{ \frac{\rm{X}}{255} - \rm{mean} }{ \rm{std} } + \rm{b}\)пјҢз„¶еҗҺзӣҙжҺҘи®Ўз®—reduce_meanеҚіеҸҜгҖӮ

    еҰӮжһңconvеҗҺйқўжҺҘзҡ„жҳҜbnпјҢconvеҸҜиғҪжІЎжңүbiasпјҢиҝҷдёӘж—¶еҖҷеҺҹжқҘзҡ„bias=0гҖӮ

    import torch
import numpy as np
from torchvision.models import resnet18
import torch.nn as nn


# жЁЎжӢҹзҒ°еәҰеӣҫиҫ“е…Ҙ
image_gray = np.random.randint(0, 256, size=(1, 224, 224))
# иөӢеҖјдёәдёүйҖҡйҒ“
image_rgb = np.repeat(image_gray, 3, axis=0)

model = resnet18(pretrained=True)
conv1 = model.conv1
model.eval()

# жҹҘзңӢеҚ·з§ҜеҸӮж•°
print(conv1)

weight = conv1.weight.data

imagenet_mean = np.array([0.485, 0.456, 0.406]).astype(np.float32)
imagenet_std = np.array([0.229, 0.224, 0.225]).astype(np.float32)


input_tensor = (image_rgb.astype(np.float32) / 255 - imagenet_mean.reshape(-1, 1, 1)) / imagenet_std.reshape(-1, 1, 1)
ori_output = conv1(torch.from_numpy(input_tensor).unsqueeze(dim=0))

new_weight = (weight / (255 * torch.from_numpy(imagenet_std.reshape(1, -1, 1, 1)))).sum(dim=1, keepdim=True)

tmp_input = np.full_like(input_tensor, 0.0)
for i, (mean, std) in enumerate(zip(imagenet_mean, imagenet_std)):
    tmp_input[i, ...] = -mean / std

# иҝҮж»ӨеҲ°paddingйғЁеҲҶеҢәеҹҹ
new_bias = conv1(torch.from_numpy(tmp_input).unsqueeze(dim=0))
# stride=(2, 2), padding=(3, 3) дёӨжӯҘжүҚиғҪдёҚеҲ©з”Ёpadding_value
new_bias = new_bias[:,:,2:-2,2:-2].mean(dim=[0, -1, -2], keepdim=False)


conv1.bias = nn.Parameter(new_bias, requires_grad=True)
conv1.weight.data = new_weight

# дҝ®ж”№еҚ·з§ҜеҸӮж•°
conv1.in_channels = 1

new_output = conv1(torch.from_numpy(image_gray.astype(np.float32)).unsqueeze(dim=0))

diff = new_output - ori_output

print(diff[:, :, 2:-2, 2:-2].abs().max()) # tensor(1.6212e-05, grad_fn=<MaxBackward1>)
